# Workflow 说明

## test-token.yml

手动触发，测试 `LZC_CLI_TOKEN` 是否配置正确，验证能否调用懒猫 API。

## update-image.yml

自动/手动检测上游新版本，构建 Docker 镜像并发布。

### 触发方式

1. **自动**：每 6 小时检查上游最新 release
2. **手动**：在 GitHub Actions 页面手动触发，可选参数：
   - `force_build`: 强制构建
   - `target_version`: 指定版本号
   - `publish_to_store`: 发布到懒猫应用商店

### 环境变量

需要配置的 Secrets:
- `LZC_CLI_TOKEN`: 懒猫 CLI Token
- `GITHUB_TOKEN`: GitHub Token（自动提供）

获取 LZC_CLI_TOKEN:
```bash
cat ~/.config/lazycat/box-config.json | python3 -c "import sys,json; d=json.load(sys.stdin); print(d.get('token',''))"
```
