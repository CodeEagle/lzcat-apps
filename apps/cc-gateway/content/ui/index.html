CC GATEWAY // 伪装协议已激活

欢迎来到 Claude Code Telemetry 净化网关。此系统提供 API 遥测的身份洗白与环境保护。

所有经过此网关的请求，将被统一抹去原始终端指纹与设备特征。

// 接入指令

在你的开发机终端中，配置以下环境变量即可黑入本网关：

# 劫持全量 Claude Code 流量指向本网关 export ANTHROPIC_BASE_URL="http://${window.location.host}" # 阻断所有侧信道遥测监控 (Datadog/GrowthBook/Version) export CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC=1 # 绕过身份拦截器（网关接管鉴权） export CLAUDE_CODE_OAUTH_TOKEN="gateway-managed" # 网关接入许可密令（需与服务端 config.yaml 配置一致） export ANTHROPIC_CUSTOM_HEADERS="Proxy-Authorization: Bearer YOUR_LOCAL_TOKEN"

// 服务端配置指南

此网关需要您的原始凭证来进行底层对话，初次启动请按以下步骤完成劫持配置：

  1. 访问该应用的文件管理，找到并编辑 /lzcapp/var/data/cc-gateway/config.yaml 文件。
  2. 在支持的且您已登录过的系统终端中键入 bash scripts/extract-token.sh 以提取高权限 OAuth refresh_token。
  3. 填入对应的 refresh_token 及您想设定的 client_tokens 接口密令。
  4. 在管理控制台重启该应用以使配置生效。
