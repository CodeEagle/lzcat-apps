# MarkItDown Web & MCP 使用教程

## 适用人群

适合希望在懒猫微服中本地运行 `MarkItDown Web & MCP`，并通过浏览器完成核心工作流的用户。

## 安装后第一步

1. 在懒猫微服中安装应用。
2. 打开 `https://markitdown.<你的盒子域名>/`。
3. 确认页面不是空白页、平台错误页或 5xx 错误页。

## 核心流程

1. 打开应用首页。
2. 按页面上的主要输入区填写或上传内容。
3. 执行主操作。
4. 确认结果区域出现可用输出。

## 验收记录

Codex Browser Use rendered MarkItDown, converted https://example.com to Markdown, and reported no console errors.

## 常见问题

- 如果页面打不开，先检查应用状态和容器日志。
- 如果页面能打开但主流程失败，检查 Browser Use 记录里的 console/network 错误。
- 如果功能需要外部账号、API key 或持久化目录，先在懒猫应用详情中补齐配置再重试。
