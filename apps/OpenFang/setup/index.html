LLM API Key Required
OpenFang 需要至少配置一个 LLM Provider 的 API Key 才能正常运行。
OpenFang requires at least one LLM Provider API Key to function.
1 获取 OpenFang API Key Get OpenFang API Key
OpenFang 启动时会自动生成一个 API Key，用于访问控制。 OpenFang generates an API Key automatically on startup for access control.

在懒猫微服中查看日志： Check the logs in LazyCat:
应用中心 → OpenFang → 查看日志 App Center → OpenFang → View logs
查找类似以下的日志行： Look for a log line like:

OpenFang API Key: of_xxxxxxxxxxxxxxxx
2 配置 LLM Provider Configure LLM Provider
在懒猫微服中配置环境变量： Configure environment variables in LazyCat:
应用中心 → OpenFang → 应用详情 → 环境变量 → 点击 展开 → 找到 openfang → 编辑 App Center → OpenFang → App Detail → Environment Variables → click Expand → find openfang → Edit
添加你的 LLM API Key，例如： Add your LLM API Key, for example:

ANTHROPIC_API_KEY=sk-ant-xxxxx

3 自动重启 Auto Restart
提交后应用会自动重启，配置生效后即可开始使用 OpenFang。 The app will automatically restart after submission. Once configured, you can start using OpenFang.
