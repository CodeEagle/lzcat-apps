# browser_acceptance_runner.py exits early with "playwright not installed in
# this environment", leaving .browser-acceptance.json unwritten and the
# verify pipeline stuck at browser_pending. Observed in stellaclaw run
# 25530542349. Pin to the same minor as the base image's browsers.
